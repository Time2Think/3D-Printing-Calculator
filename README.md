# Калькулятор стоимости 3D-печати

Готовый адаптивный статический сайт для расчета:
- себестоимости
- цены продажи
- дохода

## Как открыть локально
1. Скачайте папку или ZIP.
2. Откройте `index.html` двойным кликом.
3. Сайт будет работать даже без сервера.

## Как задеплоить бесплатно через GitHub Pages
1. Создайте новый репозиторий на GitHub.
2. Загрузите в него файл `index.html`.
3. Зайдите в `Settings` → `Pages`.
4. В блоке `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `/ (root)`
5. Нажмите `Save`.
6. Через короткое время появится ссылка вида:
   `https://username.github.io/repository-name/`

## Как загрузить на Google Drive
Google Drive удобно использовать как хранилище файла `index.html` или ZIP, чтобы переслать его другому человеку.
Но для полноценного публичного сайта лучше использовать GitHub Pages, потому что собственный веб-хостинг HTML в Google Drive давно отключен.

## Логика расчета
- Цена за 1 г = цена пластика / 1000
- Коэффициент цветов = 1 + (цвета - 1) × 0.2
- Материал = цена за 1 г × вес × коэффициент цветов
- Время = часы печати × ставка за час
- Себестоимость = материал + время
- Цена продажи = себестоимость × (1 + наценка / 100)
- Доход = цена продажи - себестоимость

## Что можно быстро поменять
В файле `index.html` можно легко изменить:
- стартовую цену пластика
- стартовую ставку за час печати
- формулу коэффициента цветов
- диапазон наценки
